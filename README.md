# SmartStudy-Scheduler

## Description
SmartStudy-Scheduler is a lightweight, Python-based utility designed to optimize academic time management. As a student aiming for global academic excellence, I developed this tool to address a common challenge: inefficient study time allocation. The program utilizes a weighted algorithm to distribute available study hours across multiple subjects based on their individual difficulty levels, ensuring that more demanding topics receive the focus they require.

## Key Features
*   **Dynamic Time Allocation:** Automatically calculates study duration per subject using a weighted distribution logic based on difficulty.
*   **User-Centric Design:** Simple and interactive Command Line Interface (CLI) for quick scheduling.
*   **Scalability:** The algorithm is designed to be easily adapted for future enhancements, such as adding priority tags or session reminders.

## How to Run
To run the SmartStudy-Scheduler on your local machine, please follow these steps:

1.  **Prerequisites:** Ensure you have Python installed on your computer. You can download it from [python.org](https://python.org).
2.  **Clone the Repository:** 
```bash
    git clone [https://github.com/YOUR_USERNAME/SmartStudy-Scheduler.git](https://github.com/YOUR_USERNAME/SmartStudy-Scheduler.git)
    ```
3.  **Navigate to the folder:**
```bash
    cd SmartStudy-Scheduler
    ```
4.  **Execute the script:**
```bash
    python study_scheduler.py
    ```
5.  **Follow the prompts:** Enter the total study hours available and the details for each subject to generate your personalized study plan.
